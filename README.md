# Simple Books Catalogue

A clean and responsive book search application built with pure JavaScript and Vite. It uses the Open Library API to help users discover books and manage their personal favorites.

**Live Demo:** [https://AntoniKra.github.io/Simple-Books-Catalogue/](https://AntoniKra.github.io/Simple-Books-Catalogue/)

## âœ¨ Key Features (Requirements Met)

- **Search Engine:** Integration with Open Library API using `fetch`.
- **Favorites Management:** Add/remove books from a personal list, persisted via `localStorage`.
- **On-the-fly Search:** Real-time results as you type, optimized with a 500ms **debounce** mechanism.
- **State Handling:** Dedicated UI for "Loading", "Connection Error", and "No Results".
- **Responsive Design:** Fully mobile-friendly layout built with pure CSS.
- **Pure JavaScript:** Built without any third-party frameworks or libraries.
